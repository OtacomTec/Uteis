Thank you for downloading SmartMenuXP.ocx.
Please feel free to use and distribute this control in your applications.
SmartMenuXP has been tested only with Visual Basic 6.

Installation
-----------
1. Register SmartMenuXP.dll
        e.g.: C:\My Folder> Regsvr32 SmartMenuXP.dll

2. Register SmartMenuXP.ocx
        e.g.: C:\My Folder> Regsvr32 SmartMenuXP.ocx

3. You can launch Visual Basic and start using this control
